A professional film journalism WordPress theme designed for cinema criticism, film theory essays, and awards prognostication.

== Description ==

LUNARFILM is a standalone WordPress theme built for serious film criticism and editorial content. Inspired by prestige publications like Bright Lights Film Journal and Criterion essays, it combines academic rigor with an engaging personal voice.

Features:
* Dark, cinematic color scheme optimized for film content
* Responsive design for all devices
* Custom post types for reviews, essays, and prognostications
* Film metadata support (director, year, genre, studio, runtime)
* Widget-ready sidebar and footer areas
* Custom navigation menus
* SEO-optimized for long-form content
* Academy Awards database integration

== Installation ==

1. Upload the theme files to `/wp-content/themes/lunarfilm/`
2. Activate the theme through the 'Appearance > Themes' menu in WordPress
3. Configure theme settings via Appearance > Customize

== Frequently Asked Questions ==

= Does this theme require a parent theme? =

No, LUNARFILM is a fully standalone theme with no parent theme dependency.

= What content types does this theme support? =

The theme supports standard WordPress posts and pages, plus custom post types for film reviews, essays, and awards prognostications.
