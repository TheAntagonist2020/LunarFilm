== Description ==

LUNARFILM is a modern, dark-themed WordPress theme designed specifically for film enthusiasts, movie blogs, and cinema-related websites. With its sleek design and comprehensive feature set, it provides everything you need to create a professional film-focused website.

= Key Features =

* Fully standalone with no parent theme dependency
* Dark color scheme optimized for film content
* Fully responsive and mobile-friendly design
* Custom logo support
* Custom background support
* Two menu locations (Primary and Footer)
* Widget-ready with sidebar and footer areas
* Featured image/post thumbnail support
* Threaded comments support
* Translation ready with text domain
* HTML5 markup
* Accessibility ready
* Clean, semantic code following WordPress coding standards

= Perfect For =

* Film review websites

* Cinema news sites
* Entertainment portals
* Personal film journals

== Installation ==

1. In your WordPress dashboard, go to Appearance > Themes > Add New
2. Click Upload Theme
3. Upload the theme ZIP file
4. Click Install Now
5. After installation, click Activate

Alternatively, you can:
1. Upload the theme folder to /wp-content/themes/ directory
2. Go to Appearance > Themes in your WordPress dashboard
3. Activate LUNARFILM

== Frequently Asked Questions ==

= Does this theme require any plugins? =

No, LUNARFILM is a fully functional standalone theme. However, you can enhance it with any WordPress plugins you choose.

= Can I customize the colors? =

Yes, the theme supports custom background colors via the WordPress Customizer. You can also modify the colors by editing the style.css file.

= Is this theme compatible with page builders? =

Yes, LUNARFILM works with popular page builders like Elementor, Beaver Builder, and others.

= Does this theme support Gutenberg? =

Yes, LUNARFILM is fully compatible with the WordPress block editor (Gutenberg).

= Can I use this theme on WordPress.com? =

Yes, LUNARFILM is designed to work with WordPress.com Business Plan and above, which allow custom theme uploads.
